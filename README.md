# Maoko Engineering — WTP Service Reports

Astro + Tailwind implementation for a responsive service reporting dashboard.

## Routes

- `/` — main dashboard with clickable service report category cards and a **New Service Report** CTA
- `/reports/new` — scaffolded Water Treatment Plant service report form

## Key Components

- `/src/layouts/BaseLayout.astro`
- `/src/components/dashboard/DashboardHeader.astro`
- `/src/components/dashboard/CategoryCard.astro`
- `/src/components/SectionCard.astro`
- `/src/components/reports/ReportSection.astro`

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
